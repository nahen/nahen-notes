トリプトファンは高タンパク質の食べ物に豊富に含まれている。  
なのでトリプトファンを多く摂りたければ、牛乳やバナナよりも肉や魚・大豆を食べたほうがいい。

## 含有量（100g中）

| 食品名 | トリプトファン含有量 |
| :-- | ---------: |
| バナナ |       10mg |
| 牛乳  |       42mg |
| 白米  |       89mg |
| 卵   |      108mg |
| 肉類  |  150～250mg |
